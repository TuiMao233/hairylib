# @tuimao/uni-use

uni 工具库, 以及修复 h5 api 不能使用

## Install

`yarn add @tuimao/uni-utils`

## Usage apis

- navigateTo
- reLaunch
- switchTab
- redirectTo
- navigateBack

- UNI_PLATFORM:   环境变量(h5|wmp-weixin|....)
- promisify:      将 uniapp api 转换为异步
- rpx2px:         将 rpx 转换为 px
- px2rpx:         将 px 转换为 rpx
