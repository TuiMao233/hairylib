# Miscellaneous

关于浏览器的工具集

## Install

`npm install @hairy/browser`

## Usage Functions

### ejectWindow

跳转到新的页面

```ts
import { ejectWindow } from '@hairy/browser'

ejectWindow('www.baidu.com')
```
### makePhoneCall

拨打号码

```ts
import { makePhoneCall } from '@hairy/browser'

makePhoneCall('173...3311')
```
