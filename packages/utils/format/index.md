# utils - format

字符串转换的集合

## Install

`npm install @hairy/utils`

## Usage

```md
# formatClearHtml(string)
> 格式化剔除字符串代码字段
# formatPrice(string)
> 格式化为价格(两位小数点)
# formatInteger(string)
> 格式化为正整数
# formatThousandBitSeparator(string)
> 格式化数字千位分隔符
# formatUnix(number, format)
> 格式化时间戳(秒)
# formatCoverPhone(string)
> 格式化隐藏手机号中间四位
# prefixZero(number, length)
> 数字位数不够时，进行前补零
# capitalizeCamelCase(string)
> 转换大写驼峰
```
