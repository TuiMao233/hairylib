# @hairy/cli

脚手架，提供创建模板与基库编译。

## Install

~~~sh
npm install @hairy/cli -g
~~~

## Usage

### 1. Create Template

~~~sh
hairy create <name>
~~~

Templates
- basic
- vue3 (never)
- uniapp (never)
- react (never)

### 2. Compile Code

~~~sh
hairy dev <path>
hairy build <path>
~~~
